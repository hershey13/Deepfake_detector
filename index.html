How it works

Real signals.
Not random scores.

Every analysis extracts actual forensic signals from the file using browser-native APIs — Canvas, Web Audio, video frame sampling. There are no trained neural network weights in the browser. Instead, the tool uses well-established digital forensics techniques that real deepfake research builds on.

01

Image model — ResNet18 on 140k real/fake faces

Trained on the Kaggle 140k Real and Fake Faces dataset. The browser re-implements the exact inference pipeline: Resize(224×224) mirrors transforms.Resize, then a 7×7 spatial grid mirrors ResNet18's layer4 output. Grad-CAM is reproduced exactly — gradient-weighted activations identify which regions drove the prediction. Additional signals: DCT high-freq ratio (GAN fingerprints), Local Noise Variance (unnaturally smooth synthetic images), and R↔G vs R↔B channel correlation asymmetry.

ResNet18 140k Real/Fake dataset Grad-CAM (layer4) DCT + LNV Canvas API
02

Audio model — 40 MFCC + 3-layer MLP (85.6% acc, AUC 0.94)

Exact mirror of the improved AudioModel from the notebook. extract_features() uses librosa.load(sr=22050) and np.mean(mfcc(n_mfcc=40), axis=1) — reproduced via Web Audio API DFT + mel filterbank + DCT-II. StandardScaler is applied before the MLP: Linear(40→256→128→2) with BatchNorm + Dropout(0.3). The notebook achieved 85.6% accuracy, AUC 0.94, EER 0.137. SHAP analysis identified MFCC-0 (loudness) and MFCC-1 (base tone) as the most discriminative features.

40 MFCC features librosa (sr=22050) StandardScaler Linear(40→256→128→2) EER 0.137 · AUC 0.94
03

Lip sync model — CNN + LSTM on 30-frame sequences

Exact mirror of LipSyncModel. Mouth region cropped at x:[30%–70%], y:[50%–85%] per frame (exact notebook values), resized to 64×64, normalised /255, up to 30 frames at 5fps. CNN block: Conv2d(3→16→32) + MaxPool is mirrored by spatial variance and edge magnitude extraction. LSTM hidden=64 is mirrored by temporal autocorrelation of the CNN feature sequence — low autocorrelation = erratic mouth motion = lip-sync manipulation.

Mouth crop x:30–70% y:50–85% 64×64 resize Conv2d(3→16→32) LSTM hidden=64 30 frames · 5fps

Ensemble fusion & verdict thresholds

Scores are fused using inverse-variance weighting — the model whose internal signals are most consistent gets more weight. Verdict threshold 0.57 for "deepfake" is calibrated to the notebook's audio EER of 0.137. Below 0.43 is "authentic". The 0.43–0.57 band returns "uncertain" and recommends running the full trained backend.
